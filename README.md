### Tree

使用Rust 重写 Tree命令，用于查看本地文件树。

目前实现以下几个命令，其他有待更新

```
本地文件树

Usage: tree [OPTIONS]

Options:
  -d, --depth <DEPTH>  文件树深度 [default: 3]
  -a, --a              展示隐藏文件
  -D, --dir            显示目录
  -F, --file           显示文件
  -p, --path           展示完全路径
  -h, --help           帮助
  -v, --verion         当前版本

```
